# 🪴 PixelPot

PixelPot is a premium, nature-themed Pomodoro timer designed to help you focus and grow your digital garden. As you work, your plant flourishes through various stages, accompanied by calming nature wallpapers and helpful auditory feedback.

## ✨ Features

- **Dynamic Plant Growth**: Your plant grows through 5 distinct stages during each 25-minute focus session.
- **Nature Wallpaper Slideshow**: High-quality nature images that crossfade smoothly every 15 seconds.
- **Sound Feedback**: Satisfying click sounds for UI interaction and an achievement bell when you finish a session.
- **Interactive Timer**: Easy-to-use controls with skip forward/backward options for testing and flexibility.
- **Premium Aesthetics**: Glassmorphism UI with dark overlays for maximum readability and a modern feel.

## 🚀 Getting Started

To run PixelPot locally, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YOUR_GITHUB_USERNAME/PixelPot.git
    cd PixelPot
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```

The app will be available at `http://localhost:5173`.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons/Graphics**: Custom SVG Illustrations
- **Images**: [Unsplash API](https://unsplash.com/developers)
